Como funciona

3 passos. Sem complicação.

Configure uma vez, funcione para sempre. Sem cadastro, sem internet constante, sem servidores.

1

Registre o lugar

Busque um endereço ou toque no mapa. Escolha um emoji e dê um nome — ou use um template pronto (ônibus, casa, tarefa).

2

Configure os raios

Defina até 5 raios com distâncias diferentes (500m a 10km). Cada raio pode vibrar ou tocar alarme alto.

3

Esqueça e viva

O app monitora em segundo plano. Quando você se aproximar, os alertas disparam — mesmo com o app fechado.

Perguntas frequentes

Dúvidas?


Precisa de internet pra funcionar?
Não. O monitoramento usa GPS nativo do celular. Você só precisa de internet para buscar endereços na hora de cadastrar um lugar.
Gasta muita bateria?
Não. O Geo Alert usa a API nativa de geofencing do iOS e Android, que é ultra-eficiente. Sem polling de GPS constante.
Funciona com o app fechado?
Sim! O geofencing roda em segundo plano. Mesmo com o app fechado, os alertas disparam normalmente.
Meus dados vão para algum servidor?
Não. Todos os dados ficam no seu celular. Não temos servidores, não coletamos nada. Privacidade total.
Quantos lugares posso monitorar?
No Android, sem limite prático. No iOS, o sistema permite até 20 geofences simultâneos (limitação da Apple).
Posso compartilhar o APK com amigos?
Sim! O APK Android pode ser enviado diretamente. Para iOS, o app precisa ser instalado via Xcode ou TestFlight.
