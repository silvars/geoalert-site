Como funciona

3 passos. Sem complicação.

Configure uma vez, funcione para sempre. Sem cadastro, sem internet constante, sem servidores.

1

Registre o lugar

Busque um endereço ou toque no mapa. Escolha um emoji e dê um nome — ou use um template pronto.

2

Configure os alertas

Defina até 5 alertas com distâncias diferentes (500m a 10km). Cada um pode vibrar ou tocar alarme alto.

3

Esqueça e viva

O app monitora em segundo plano. Quando você se aproximar, os alertas disparam — mesmo com o app fechado.

Perguntas frequentes

Dúvidas?


Precisa de internet pra funcionar?
Não. O monitoramento usa o GPS do seu celular. Você só precisa de internet para buscar endereços na hora de cadastrar um lugar.
Gasta muita bateria?
Não. O Geo Alert usa o sistema nativo do celular para monitorar localização, que é ultra-eficiente.
Funciona com o app fechado?
Sim! O monitoramento roda em segundo plano. Mesmo com o app fechado, os alertas disparam normalmente.
Meus dados vão para algum servidor?
Não. Todos os dados ficam no seu celular. Não temos servidores, não coletamos nada. Privacidade total.
Quantos lugares posso monitorar?
No Android, sem limite prático. No iOS, o sistema permite até 20 locais simultâneos (limitação da Apple).
Posso compartilhar o APK com amigos?
Sim! O APK Android pode ser enviado diretamente. Para iOS, o app precisa ser instalado via TestFlight.
